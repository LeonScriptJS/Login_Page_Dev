# Modern Login & Registration Page

A modern and responsive authentication UI built with **HTML, CSS, and JavaScript**.
This project features smooth transitions between **Sign In** and **Sign Up** panels, social media login icons, form validation, and a clean premium-style interface.

## How It Works

The project uses a main container that switches between:

* **Sign In Mode**
* **Sign Up Mode**

JavaScript handles the toggle functionality by adding or removing active classes from the container.

---

## Form Validation

The forms currently use built-in HTML validation, such as:

* `required`
* `minlength`
* `maxlength`
* `type="email"`

This helps ensure cleaner and more secure user input.
